The INA driver provides API to configure and measure current using I2C interface present on the board.

The driver takes care of configuring and reading current values from INA Sensor

Features Supported

  • API to configure INA sensor using I2C interface
  • Provides API to return current values from the INA

SysConfig Features

Note
It is strongly recommend to use SysConfig where it is available instead of using direct SW API calls. This will help simplify the SW application and also catch common mistakes early in the development cycle.
  • Option to configure INA registers
  • Supported INA devices
    • INA228

N/A

Important Usage Guidelines

  • There are four power rails whose current can be measured using corresponding INA sensors
  • The INA configurations can be done using Sysconfig
  • By default, Sysconfig chooses appropraite Target address for default INA sensor associated with each rail on the EVM
  • Currently, 3.3V rail INA sensor value is not measured
