DPU_Aoa2dProc_HWA_Doppler_Fft_Cfg Struct Reference

Detailed Description

Configuration parameters for each HWA param set performing the Doppler FFTs and mappng output into row-col array as input to azimuth/elevation FFTs.

Input antenna format to Doppler FFTs per range bin is X[antInd][chirpInd] Output antenna format is Y[dopplerInd][rowInd][colInd] HWA param set can perform only fixed source and destination increment (B dimension)
