About the Gesture classifier

The gesture classifier uses an ANN (Artificial Neural Network) for gesture classification along with 2-hidden layers. The standard non-linearity function of ReLU is used in between the layer outputs and the output layer uses softmax function to get the prediction probability. The input to the ANN is a set of features (based on user selection, currently we use elevation weighted mean, doppler average and range average) accumulated across n-frames (say for example in gesture demo its currently set to 15).

Note that the input applied to standard normalization before giving as input to ANN.

3-features, 15frames accumulated across input to 2-layer ANN with 2-outputs
