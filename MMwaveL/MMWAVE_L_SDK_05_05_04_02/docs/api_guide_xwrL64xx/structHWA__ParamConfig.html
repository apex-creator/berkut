◆ dmaTriggerSrc

uint8_t HWA_ParamConfig::dmaTriggerSrc

DMA channel number to be monitored for dma based trigger mode. this field is used only when HWA_ParamConfig::triggerMode is set to HWA_TRIG_MODE_DMA Sets the DMA2ACC_CHANNEL_TRIGSRC bits in PARAM1_0

◆ accelMode

uint8_t HWA_ParamConfig::accelMode

2 bit value: See HWA_ACCELMODE macros for correct values. Configures the mode of operation of the accelerator for the current parameter-set. Sets the ACCEL_MODE bits of PARAM1_0
